# Context: Nomaq

## Overview
Nomaq is a high-fidelity mobile-first React/Next.js travel booking web application.
Key elements:
- Vertical scrolling TikTok-style feed (Feed Ispirazionale)
- Drops section: Simulated price drop alerts with pop-up/toast notification
- Salvati section: User can favorite/save routes from the feed
- Profilo section: User profile view
- Waitlist Landing Page: Separate route (`/waitlist`) with email subscription form and shareability feature ("Flexa il tuo Drop")
- Design System: White background, Electric Orange buttons, Anthracite Grey text, Glassmorphism elements.
